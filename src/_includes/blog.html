This page (blog.html) sits in the _include folder and is used in the front matter of blog/index.html ( to which the menu item links) which in turn iterates over the posts collection [collections.post] taking a few elements only of each post ( URL, date, author and description ( explore truncate for height consistency)). The data is held in a ".card" class which is styled as fairly basic cards here.

Each blog post uses the post template as a layout ( assigned in posts.jason - an example of a directory data object which applies to all files in the same directory - cool) )

The post tempplate has a class of ".full-blog" which styles it

A class of ".in-post" is user created within each blog post md file ( it is possible to mix in html elements to a markdown file pr just write the blog in html too of course)

Styling

The following cards could be styled completely differently. If you wish to do this then go to blog/index.html in src folder to add other css classes and generally rearrange the html for a different look
