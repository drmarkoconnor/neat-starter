11ty Collections

Take any existing tagName and iterate over collections.tagName with liquid or nunjucks template code

Result for tagName = "test"

    {% if collections.test%} {% for item in collections.test %}
  • {{item.data.title}}
    • {% endfor %} {% else %}There are no posts with the tag test. Why not add one to see what happens here {% endif %}

Result for tagName = "post"

    {% for item in collections.post %}
  • {{item.data.title}}
    • {% endfor %}

Other data

From authors.yaml in the _data directory. By iterating over the authors.yaml file it is possible to pick out just the property you want ( in this case name) => authors.authors.name

    {% for item in authors.authors %}
  • {{ item.name }}
    • {% endfor %}

It is possible to use the admin/config.yaml file to set up the Netlify CMS editor to read and write to the file and therefore allow authenticated web users to create more examples. This is the same for the quicklinks.yaml and navigation.yaml files which can be used as examples for other use cases too

Special all collection

By default Eleventy puts all content (independent of whether or not it has any assigned tags) into a collections.all Collection. This allows an iteration over all content inside of a template.

    {%- for post in collections.all -%}
  • {{ post.url }}
    • {%- endfor -%}
