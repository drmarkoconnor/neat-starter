This page iterates over a team.json database (in the _data/ directory) creating team members with 4 properties, Name, Role, Tel, Email

The front matter contains a height: 10vh key value pair which is used to set the in line style property of the cards ( which would otherwise be set to 40vh in the main css stylesheet.)

The in line style has a higher specificty so this is a good trick. See blog page for same class element but using the height:40vh setting from styles.css
